## 课程目标
- **开发能力**：了解并掌握一种开源的大模型系统，并且有能力在之上进行二次开发和应用。
- **科研能力**：通过参与老师，博士的研究工作，提升自己的科研能力。
- **表达能力**：通过参与论文撰写，分享相关工作（包括制作PPT），提升自己的表达能力。
- **项目经验**：参与并完成一个大模型项目，投稿，甚至发表相关论文。
