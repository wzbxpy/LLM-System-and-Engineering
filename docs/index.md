## 课程简介
大模型系统与工程课程旨在介绍大规模语言模型（LLM）的基本原理、推理框架以及实际应用。通过本课程的学习，学生将掌握LLM的基本概念、推理流程以及如何使用vLLM等推理框架进行模型部署和应用。
